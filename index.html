Front-End Web Technologies

A web browser displays a web page using

  • HTML
  • CSS
  • Javascript (ECMAScript)

HTML

The hypertext markup language uses tags for different parts of the content

For example 'p' is a paragraph, 'ul' is an unordered list

Most content needs an open tag and a close tag

There are a few 'self-close' tags, e.g. 'br'
, which is a line break

Tags may have attributes

A tag and all its content is called an element

CSS

Cascading Style sheets bring style to the content

  2. Class Style
  3. Tag Style
